西藏拉萨：“望果”赛马正当时

新华网 | 2024年07月24日 06:34:11
原标题：西藏拉萨：“望果”赛马正当时

央视新闻： 新华社拉萨7月23日电（刘洲鹏、格桑雍珍）近日，西藏各地迎来望果节。望果节是西藏传统的丰收节，也是国家级非物质文化遗产，具体日期随各地庄稼成熟的时间而变化。望果节期间，按照习俗，各地都会举办赛马比赛。

23日，在拉萨林周县甘旦曲果镇江角村的赛马场上，选手边玛益丹穿着轻便的马术运动服，上身贴紧棕色马背，右手的马鞭在空中打着旋，和他的爱马进行最后冲刺。

当日比赛中，上百名选手根据参赛马匹的年龄被分入不同的组别。边玛益丹在5岁及以上的“大马”组比赛中夺冠。他和他的赛马来自那曲市嘉黎县，这是他第二次参加林周县的赛马比赛。“也有很多人到我们那里去参加赛马比赛，我们牧区之间常常有赛马交流。”他说。
